# AI-Quant-Framework
> AI驱动的量化交易全栈框架 | OpenClaw 生态专属 Skill
> 
> ## 🚀 功能特性
> - ✅ **三引擎架构**：回测/实测/实盘一体化
> - ✅ **双交易所支持**：币安 + OKX 无缝切换
> - ✅ **可视化K线**：实时行情图表，交易信号打点标记
> - ✅ **操作日志**：全流程记录，可追溯可复盘
> - ✅ **策略热加载**：无需重启，动态加载策略文件
> - ✅ **风险控制**：内置止损止盈，仓位管理
> - ✅ **多周期支持**：1分钟到周线全周期覆盖
> - ✅ **OpenClaw 原生集成**：可作为 OpenClaw Skill 直接运行，无缝接入 OpenClaw 生态，支持多 Agent 协作、定时任务、跨渠道消息通知、远程控制等能力，可直接部署为 OpenClaw 量化交易技能模块
> 
> ## 📦 快速开始
> ### 环境要求
> Python 3.10+
> 
> ### 安装依赖
> ```bash
> pip install -r requirements.txt
> ```
> 
> ### 运行程序
> ```bash
> python 币安量化框架_升级版.py
> ```
> 
> ### 作为 OpenClaw Skill 运行
> 1. 将本项目放到 OpenClaw skills 目录
> 2. 在 OpenClaw 配置中启用量化交易技能
> 3. 直接通过自然语言即可调用："帮我跑一下双均线策略回测"、"查看当前实盘收益"
> 
> ### 使用流程
> 1. 配置交易所API（可选，实盘需要）
> 2. 选择交易对和时间周期
> 3. 加载策略文件（需实现`trade_signal`函数）
> 4. 选择引擎模式，点击启动
>
